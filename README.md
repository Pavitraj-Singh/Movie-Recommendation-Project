# Movie Recommendation System

This is a web-based movie recommender system built using Python and Streamlit. It uses a content-based filtering approach to suggest similar movies and displays posters using the TMDB API.

## Features

- Recommends top 5 similar movies
- Fetches movie posters via TMDB API
- Streamlit UI with background styling

## Files

- `app.py` – Main Streamlit application
- `movie_list.pkl` – Pickled DataFrame of movies
- `similarity.pkl` – Similarity matrix for recommendations
- `requirements.txt` – Python dependencies
